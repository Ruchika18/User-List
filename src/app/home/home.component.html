How it Works?

This is User List Application in which you can Add, Edit, Delete and View the User Information.

  • User can View User List along with Name, City and Department.
  • User can perform sorting, filtering on each column.
  • It also supports pagination

User can Add / Delete / Edit the Records as below:

  • Click on add symbol at top right to add the new record.
  • Click on edit symbol for editing a existing user record.
  • Click on Delete symbol for deleting an record
